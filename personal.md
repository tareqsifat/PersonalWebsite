## Design Instructions

- **Color palette:** Keep it minimal. Dark navy or charcoal for headers, white background, one accent color (pick a clean blue or teal). Black body text.
- **Typography:** System fonts only — no Google Fonts imports. Use: `font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Arial, sans-serif`
- **Layout:** Max content width 960px, centered. Generous padding and whitespace.
- **Cards:** Subtle border or box-shadow. Rounded corners (4–8px). No heavy shadows.
- **Mobile:** Stack all columns to single column below 768px. Navigation collapses cleanly.
- **No JavaScript** unless absolutely necessary for a mobile nav toggle — if needed, use minimal vanilla JS inline only.
