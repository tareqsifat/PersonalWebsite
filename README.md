# Portfolio Website

Personal portfolio for Md. Tareq Aziz Sifate.

## How to host on GitHub Pages
1. Create a new GitHub repository named `username.github.io` (replace username with your GitHub username)
2. Push all files to the main branch
3. Go to Settings → Pages → Source: Deploy from branch → Branch: main → / (root)
4. Your site will be live at https://username.github.io within a few minutes

## Files
- index.html — main portfolio page
- style.css — all styles
- photo.jpg — replace with your actual photo (same filename)
